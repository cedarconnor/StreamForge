Quickstart — how to run
  1. Pick Input Type + Source (Webcam 0, or Browse… to a video File — it loops).
  2. Pick Backend: FLUX = per-frame restyle; SANA = temporal (launch with start-sana.bat).
  3. Write a Prompt for the look you want.
  4. Validate — opens the source once and shows the input/output sizes.
  5. Start — first start loads the model (~10–30 s), then the Output preview goes live.
  6. Tune the Live Control sliders while running — no restart needed.

Recommended first run: File/Webcam · Backend FLUX · Aspect Auto preserve · Preset BALANCED · Steps 4 · Ref ~0.55. In Run Health, Emitted should climb and Jitter stay low (single-digit ms).
